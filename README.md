# Task Manager Web App

A simple and responsive web application for managing daily tasks.

## Description
This project helps users organize their tasks by adding, editing, deleting, and marking tasks as completed.

## Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub

## Features
- Add new tasks
- Delete tasks
- Mark tasks as done
- Responsive design

## How to Run
1. Download the project
2. Open index.html in browser
